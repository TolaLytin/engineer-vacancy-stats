# Техническое задание: Аналитический дашборд для инженеров-программистов

## 1. Цель проекта
Создание веб-платформы для анализа рынка труда инженеров-программистов на основе данных HeadHunter. Система предоставляет статистику зарплат, спроса на профессию, регионального распределения и требований к навыкам.

## 2. Роли пользователей
- **Гость**: Просмотр статистики, графиков, таблиц и актуальных вакансий.
- **Администратор**: Управление контентом, загрузка новых данных, обновление графиков.

## 3. Модели данных
**Homepage** - Главная страница с описанием профессии
- header (Text) - Заголовок
- summary (Text) - Описание профессии
- banner_image (Image) - Баннерное изображение
- extra_images (ForeignKey to HomepageImage) - Дополнительные изображения

**StatisticalData** - Основные статистические данные
- yearly_salary_graph (Image) - График зарплат по годам
- yearly_salary_table (Text) - Таблица зарплат по годам
- yearly_vacancy_graph (Image) - График вакансий по годам
- yearly_vacancy_table (Text) - Таблица вакансий по годам
- city_salary_graph (Image) - График зарплат по городам
- city_salary_table (Text) - Таблица зарплат по городам
- city_vacancy_graph (Image) - График вакансий по городам
- city_vacancy_table (Text) - Таблица вакансий по городам
- skill_top20_graph (Image) - График топ-20 навыков
- skill_top20_table (Text) - Таблица топ-20 навыков

**Skillset** - Наборы навыков
- table_header (Text) - Заголовок таблицы
- skill_table (Text) - Таблица с навыками
- skill_graph (Image) - График навыков

**JobDemand** - Данные о спросе на профессию
- salary_graph (Image) - График зарплат
- vacancy_graph (Image) - График вакансий
- salary_table (Text) - Таблица зарплат
- vacancy_table (Text) - Таблица вакансий

**RegionData** - Региональные данные
- city_salary_graph (Image) - График зарплат по городам
- city_vacancy_graph (Image) - График вакансий по городам
- city_salary_table (Text) - Таблица зарплат по городам
- city_vacancy_table (Text) - Таблица вакансий по городам

## 4. Ключевой функционал
* Просмотр статистики по годам: динамика зарплат и количества вакансий
* Анализ регионального распределения: зарплаты и вакансии по городам
* Изучение востребованных навыков: топ-20 требований работодателей
* Просмотр актуальных вакансий с HeadHunter в реальном времени
* Визуализация данных через интерактивные графики

## 5. Внешние интеграции и аналитика
- **API HeadHunter**: Получение актуальных вакансий в реальном времени
- **Pandas**: Обработка и анализ больших объемов данных о вакансиях
- **Matplotlib**: Генерация графиков и визуализация статистики
- **API ЦБ РФ**: Конвертация валют для расчета зарплат в рублях
- **BeautifulSoup**: Очистка HTML-контента из описаний вакансий

## 6. Изменения в ходе реализации
- Добавлена многопроцессорная обработка данных для ускорения анализа
- Реализована система кэширования результатов парсинга
- Добавлены контекст-процессоры для передачи глобальных констант
- Улучшена обработка ошибок при работе с внешними API
