# Software Engineer Job Dashboard

Веб-приложение для анализа рынка труда инженеров-программистов на основе данных HeadHunter. Система предоставляет статистику зарплат, спроса на профессию, регионального распределения и требований к навыкам.

**Ссылка на рабочий проект:** [https://.pythonanywhere.com]

## Технологии
* **Python 3.x**
* **Django 5.1**
* **Pandas** - обработка данных
* **Matplotlib** - визуализация
* **Requests** - работа с API
* **BeautifulSoup4** - парсинг HTML
* **SQLite** - база данных
* **Bootstrap 5** - фронтенд
